Why E-Commerce?

I chose to build an e-commerce store because it's the perfect project to learn JavaScript fundamentals. Working with arrays, objects, local storage, and DOM manipulation felt challenging at first, but seeing the cart work made it all worth it. This project gave me confidence to build more complex applications.

Features I Built

Shopping Cart

Full cart functionality - add items, update quantities, remove products, and see totals update in real-time.

Product Filtering

Filter products by category, sort by price, and search for specific items.

Local Storage

Cart items persist even after refreshing the page - learned how to use localStorage.

Search

Real-time search that filters products as you type.
